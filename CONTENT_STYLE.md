# Аналіз стилю 001k.Trade → Адаптація для ThreeWordsDailyChat

## Що робить 001k.Trade (116K підписників)

### Формула поста:
```
[Дата/widget з даними]     ← цифри, що виглядають авторитетно
[1-3 речення коментарю]    ← коротко, свій голос, сленг
[CTA до Mini App]          ← "Бонус в нашому міні аппе"
```

### Стиль тексту:
- **Тон**: як від друга-трейдера, не від компанії
- **Довжина**: 1-3 речення MAX
- **Мова**: мікс рос/укр + англ терміни ("80k soon", "NY open")
- **Сленг**: "толпа переобувається", "бичка" — insider language
- **Емодзі**: 1-2, не більше, і тільки що підсилюють
- **Дата**: не пише, але відмітка часу є
- **Engagement**: 6.7K переглядів на пост

### Чому це органічно:
1. Дані + особиста думка = не реклама, а "ділюся"
2. Короткий текст = читається повністю
3. Insider tone = "я один з вас"
4. Mini App button внизу = не кричить "купи"

## Адаптація для ThreeWordsDaily

### Наша формула поста:

```
[Word widget або факт-цифра]   ← "300 слів = 65% розмовної англійської"
[1-2 речення особисто]         ← як від людини, не від бота
[м'який CTA]                   ← кнопка до Mini App або @бот
```

### Шаблони постів у стилі 001k.Trade:

**Тип 1 — Слово дня:**
```
resilient /rɪˈzɪliənt/ — стійкий

Найчастіше чую від HR на співбесідах.
"Are you resilient?" — а ти? 🤔

[Відкрити урок →]
```

**Тип 2 — Статистика:**
```
300 слів покривають 65% розмовної англійської.
600 — вже 85%.

Ми на 3 словах на день. До 300 — 100 днів.
Починаємо? 📈

[Старт →]
```

**Тип 3 — Idiom:**
```
"Break a leg" — не зламай ногу, а удачі 😅

Англійська буквально знущається над перекладачами.
Ще 49 таких виразів → у боті.
```

**Тип 4 — Порівняння:**
```
Duolingo: 20 хв на день, 365 нотифікацій, 0 розмов.

@ThreeWordsDailyChat: 3 слова, 1 хвилина, AI пояснює.

Обидва безкоштовно. Але одне — не дратує 🙂
```

**Тип 5 — Факт п'ятниці:**
```
Англійське слово "set" має 430 різних значень.

Мова серйозно ненавидить нас 💀

3 нормальних слова щодня → @ThreeWordsDailyChat
```

### Відмінності від поточного стилю:
| Зараз | 001k.Trade стиль |
|-------|-----------------|
| "Знайшов безкоштовний бот..." | "resilient — ось чому тебе не беруть на роботу" |
| 5-7 рядків | 2-3 рядки |
| Пояснення | Провокація/факт |

## Mini App стратегія (як у Alpha A01K)

### Що повинно бути в нашому Mini App:
1. **Daily Lesson** — слова дня з кнопками
2. **Tasks** — "вивчи 5 слів → отримай бейдж"
3. **Streak** — вогонь та нагороди
4. **Leaderboard** — топ тижня
5. **Gifts** — Telegram Stars за streak 30+ днів

### Кнопка в каналі:
Замість тексту "спробуй бот" — кнопка **[📚 Урок дня]** яка відкриває Mini App.
