## 🤖 БОТИ (7 штук)

### 1. `learning_bot.py` — Головний навчальний бот
- **Telegram:** @YourBot_prod_bot
- **Що робить:** Приймає юзерів, веде уроки, рахує XP і стріки
- **Запускається:** Автоматично через macOS LaunchAgent
- **Розклад:** Постійно активний, відповідає на команди

### 2. `content_publisher.py` — Генератор контенту
- **Telegram:** @YourBot_test_bot
- **Що робить:** Публікує в групу щодня за розкладом:
  - 09:00 — Слово дня (з перекладом і прикладом)
  - 13:00 — Вікторина (quiz)
  - 13:00 — Ідіома тижня
  - 16:00 — Граматична порада
  - 19:00 — Міністорія для читання
  - 20:00 — Мотиваційний пост
  - 21:00 — Speaking challenge (через speak_bot)
- **Запускається:** Автоматично, постійно працює

### 3. `teacher_bot.py` — Бот-вчитель (в групі)
- **Telegram:** @YourBrand_group_teacher_bot
- **Що робить:** Відповідає на питання в групі, пояснює граматику, допомагає юзерам

### 4. `analyst_bot.py` — Аналітика (тільки для тебе)
- **Telegram:** @Filechecksbot (приватний чат з тобою)
- **Що робить:**
  - Кожен понеділок 09:00 → **автоматично** шле тижневий дайджест
  - 08:00 і 20:00 → короткий пульс активності
  - Команди: `/stats`, `/retention`, `/top`, `/weekly`, `/ideas`, `/growth`
- **Це твій особистий інструмент управління**

### 5. `marketer_bot.py` — Маркетинг і реферали
- **Telegram:** @Myadsformebot
- **Що робить:**
  - Реферальні посилання (кожен юзер може запросити друзів)
  - Відстежує хто кого привів
  - Команда `/invite` — юзер отримує своє реф. посилання

### 6. `speak_bot.py` — Тренер вимови
- **Telegram:** @SpeakBetterrbot
- **Що робить:**
  - IPA транскрипція будь-якого слова
  - Пояснює типові помилки українців
  - 21:00 — щоденний speaking challenge в групі
  - Команди: `/pronounce слово`, `/say слово`

### 7. `watchdog.py` — Сторожовий пес 🐕
- **Telegram:** використовує токен analyst_bot для алертів
- **Що робить:** Кожні 5 хвилин перевіряє чи всі 7 сервісів живі
- **Якщо щось впало:** Автоматично рестартує + надсилає тобі алерт в Telegram
- **Запускається:** Автоматично через LaunchAgent


## 🆘 Якщо щось зламалось

| Проблема | Рішення |
|----------|---------|
| Бот не відповідає | `./run.sh restart learning_bot` |
| Немає контенту в групі | `./run.sh restart content_publisher` |
| Помилка в логах | `tail -50 logs/learning_bot.log` |
| Все впало | `./run.sh stop && ./run.sh start` |
| Watchdog не надсилає алерти | Перевір `ADMIN_CHAT_ID` в `.env` |
