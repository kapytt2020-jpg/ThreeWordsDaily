# Корисні GitHub репозиторії для ThreeWordsDaily

## 🔑 Критично важливі

### 1. Telethon — постинг в групи від USER акаунту
- **Repo**: https://github.com/LonamiWebs/Telethon
- **Stars**: 11.9K ⭐
- **Навіщо**: Бот НЕ МОЖЕ сам писати в групи де він не адмін. Telethon дозволяє автоматизувати ЮЗЕР-акаунт і постити куди завгодно
- **Встановлення**: `pip install telethon`

### 2. python-telegram-bot — основний фреймворк
- **Repo**: https://github.com/python-telegram-bot/python-telegram-bot
- **Stars**: 25K ⭐
- **Навіщо**: Основа для нашого Reply Bot (@Clickecombot)

## 📣 Промо і ріст

### 3. ChannelAutoPost — авто-репост між каналами
- **Repo**: https://github.com/xditya/ChannelAutoPost
- **Stars**: 153 ⭐

### 4. refbot — реферальна система
- **Repo**: https://github.com/Khuzha/refbot

### 5. Python-Telegram-Broadcast-Bot — масова розсилка
