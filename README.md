# ThreeWordsDaily

Automated Telegram bot for daily English language learning with AI-powered content generation, user progression tracking, and engagement features. Built with n8n workflow automation.

## Overview

ThreeWordsDaily is a comprehensive educational Telegram bot designed to help users learn three English words daily. The bot leverages OpenAI's GPT-4 for content generation and uses n8n for workflow automation.

### Key Features

- **Daily Content Delivery**: Automatic messages at 9:00 AM and 7:00 PM
- - **AI-Powered Generation**: Uses OpenAI GPT-4 for quiz creation and responses
  - - **User Progression Tracking**: Monitors user levels, streaks, and mistakes
    - - **Leaderboard System**: Weekly leaderboards showing top performers
      - - **Interactive Quizzes**: AI-generated quizzes with mistake analysis
        - - **TTS Support**: Text-to-speech voice messages
          - - **Admin Panel**: Approval system for new content
            - - **Multi-Bot Setup**: Support for test, production, and specialized bots
              - - **Growth Opportunities**: Daily search for brand partnership opportunities
                - - **Engagement Features**: Streak tracking, achievement milestones, bonus points

                                                                                                                                                                                                                         - ## Known Issues & Solutions
                                                                                                                                                                                                                        
                                                                                                                                                                                                                         - Based on the workflow complexity, potential challenges and solutions:
                                                                                                                                                                                                                        
                                                                                                                                                                                                                         - ### Common Issues
                                                                                                                                                                                                                        
                                                                                                                                                                                                                         - 1. **Message Formatting Complexity**
                                                                                                                                                                                                                           2.    - Issue: Complex HTML/Markdown formatting in AI-generated messages
                                                                                                                                                                                                                                 -    - Solution: Implement message formatting validation before sending
                                                                                                                                                                                                                                  
                                                                                                                                                                                                                                      - 2. **Admin Approval Timing**
                                                                                                                                                                                                                                        3.    - Issue: 25-minute wait window for admin approval may timeout
                                                                                                                                                                                                                                              -    - Solution: Implement webhook-based immediate notifications with fallback scheduling
                                                                                                                                                                                                                                               
                                                                                                                                                                                                                                                   - 3. **API Rate Limiting**
                                                                                                                                                                                                                                                     4.    - Issue: Multiple simultaneous API calls to OpenAI and Telegram
                                                                                                                                                                                                                                                           -    - Solution: Implement queue-based processing and rate limiting per bot
                                                                                                                                                                                                                                                            
                                                                                                                                                                                                                                                                - 4. **Data Consistency**
                                                                                                                                                                                                                                                                  5.    - Issue: User data spread across multiple sheets and operations
                                                                                                                                                                                                                                                                        -    - Solution: Implement transaction-like behavior for multi-step updates
                                                                                                                                                                                                                                                                         
                                                                                                                                                                                                                                                                             - 5. **Duplicate Processing**
                                                                                                                                                                                                                                                                               6.    - Issue: Same messages processed multiple times
                                                                                                                                                                                                                                                                                     -    - Solution: Implement message ID tracking and deduplication
                                                                                                                                                                                                                                                                                      
                                                                                                                                                                                                                                                                                          - ### Long-Standing Challenges
                                                                                                                                                                                                                                                                                      
                                                                                                                                                                                                                                                                                          - 1. **Multi-Bot Synchronization**: Keeping state consistent across 4 different bots
